Let's get into it.

Why these exist

Most job searches fall apart for the same boring reason: nothing remembers anything. You apply, you hear nothing, you move on, and the next application starts from zero. There's no memory, so there's no learning. You repeat the same mistakes for months without noticing.

The idea behind these skills is simple. Treat every application as a data point. A rejection and a callback are both information. If you capture what you led with, then later capture what happened, you slowly learn which roles, which framings, and which keywords actually get you in the door. The system gets a little sharper every week.

Every application is a labelled data point. Most people just throw the label away.

The whole point: that dotted line. Every loop makes the next one sharper.

The second idea is about truth. These never invent a metric or a job you didn't have. They take your real achievements and reframe and reorder them to fit the role in front of you. If a job wants something you don't have, the system flags it as a gap honestly, so you can speak to it in the interview instead of getting caught out. (Trust me, that's the better trade.)

What each skill does

Skill 1

job-search-os

Your search command centre. It interviews you once and builds living files: who you are, what you're targeting, where to look, what you're learning. Then it finds roles, checks each listing is actually live so you stop wasting clicks on dead links, scores them, and tracks them. Over time it learns your patterns and tells you what's working.

Skill 2

resume-tailor

Your resume writer. Give it a job description as text, a link, or a file, and it builds a tailored resume that mirrors the role's language, leads with your strongest-matching proof, and reuses phrasing that already worked. Ask after, and it writes a matching cover letter in your voice. Resume first, always.

They're better together. job-search-os builds and maintains your files. resume-tailor reads those same files and writes outcomes back into them. Install both and the loop closes. But each one also works perfectly fine on its own, so don't stress if you only want one to start.

How to use them

Honest setup note first: these work best in Cowork with a folder connected, or in a Claude Project. That's because they rely on those living files sticking around between chats. In a normal one-off chat they still run, but nothing carries over, so you lose the memory that makes them useful. Connect a folder if you can.

Now the steps.

1

Install both skills. Click each .skill file I sent you and hit Save skill. That's it, they're available to use.

2

Set up your system. Start a chat and say "help me set up my job search." It asks a focused batch of questions about your background, what you're targeting, and your biggest wins with real numbers. Give it proper metrics here. The more honest detail you put in, the better everything downstream gets. (Vague in, vague out.)

3

Find roles. Say "find roles for me" or "what should I apply to." It searches, weeds out the dead listings, scores what's left, and tells you why each one fits, so you can push back when it gets you wrong.

4

Tailor your resume. When you find one you like, paste the job description and say "tailor my resume for this." You get a resume back, plus a note on the angle it led with and any gaps to prep for. Ask for a cover letter only once the resume feels right.

5

Feed it the outcomes. This is the part everyone skips, and it's the whole point. Every time you hear back (ghosted, rejected, recruiter call, interview), tell it. It logs the signal and slowly figures out what's working for you. After a few weeks, your resumes get noticeably sharper because they're learning from real results, not guesses.

One quiet rule it follows: would a tired recruiter, skimming for 20 seconds, instantly see why you fit this exact role? If not, it re-orders until they would. That's the bar.

How to customise it

Good news: you barely have to. The customisation mostly happens by using it. But if you want to take the wheel, here's where everything lives.

After setup, you'll have a folder of plain markdown files. They're yours, and you can open and edit any of them by hand anytime. The ones worth knowing:

  • profile.md is your achievement bank. This is the one to invest in. Every metric and bullet comes from here, so if a number is wrong or a win is missing, fix it here and everything updates.
  • resume-style-guide.md is how your resume gets built and written: the voice, tone, font, layout, spacing, and do's and don'ts. Point it at your best existing resume as the template and it matches that look every time.
  • search-config.md is what you're hunting for: titles, locations, salary floor, the domains you want and the ones you don't. Edit this whenever your search shifts. The next run reads it fresh.
  • keyword-ledger.md is the learning brain: which framings led to callbacks versus auto-rejections. You don't really edit this one, you let it fill up. But it's fascinating to read back after a month.

Two simple ways to steer it:

  1. Just tell it. Say "stop showing me on-site roles" or "I want to lean more into X." It updates the config and remembers. Your direct feedback always outranks its own guesses.
  2. Edit the file. If you'd rather, open search-config.md or profile.md and change it yourself. Both routes end up in the same place.
Heads up, these start blank on purpose. The first run feels like an interview, not magic. That's the setup tax. Once your files exist, every run after is fast. Don't judge it by minute one.

Wrapping up

That's the whole thing. Not complicated, just a system that remembers, which turns out to be the part most job hunts are missing.

If you only remember five things:

  • Install both by saving the two .skill files.
  • Connect a folder (Cowork or a Project) so it keeps memory.
  • Start with "help me set up my job search" and give it real numbers.
  • Use it by pasting a job description and saying "tailor my resume for this."
  • Feed it outcomes every single time you hear back. That's what makes it smart.

Do that, and in a few weeks you'll have a job search that's actually working for you instead of you working it. Go get it. 💪
